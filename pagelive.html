Why PageLive?

Using a screen reader on Gemini means constant, manual switching between input, responses, and side panels. It slows down your workflow. PageLive eliminates the friction. It monitors the page and puts crucial actions—like navigating to the input or reviewing the last response—right at your fingertips.

Streamline Your Gemini Workflow

Action Shortcut Why You'll Use It
Jump to Input Shift + Esc Instantly start typing your next prompt.
Review Last Response Ctrl + Shift + Enter Quickly re-read the Gemini's last response.
Toggle Page Info dialog Ctrl + / Get a quick overview of the current page and access all PageLive keyboard shortcuts.
Toggle Content Map dialog Ctrl + M View a quick **snapshot** of the active chat history and jump directly to any previous prompt or response.
Jump to Chat List Press Enter on the chat title (Heading 1) in the Page Info dialog (Ctrl + /) Quickly move focus from the chat window to the list of previous chats.
Start New Chat Ctrl + Alt + O Clear the deck and begin fresh in a single press.
Delete active Chat Ctrl + Shift + Backspace Trigger Gemini's confirmation dialog to delete the active chat. User needs to select and click the Delete button to confirm

Content Map dialog: Your Conversation Roadmap

The Content Map dialog (Ctrl + M) gives you a brief, sequential summary of everything that has been asked and answered in the current chat. This helps you quickly regain context and understand the flow of a long chat.

  • Quick Jump: Each summary entry in the dialog is a button. Pressing Enter on any prompt or response will close the map and immediately move your focus to that specific item on the main page.
