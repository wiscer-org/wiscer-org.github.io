PageLive for Gemini

Keyboard-first workflows and accessibility-friendly announcements to make Gemini navigation faster, clearer, and more predictable.

← Go to PageLive Home

For Power Users

Focus Chat Input

Press Shift + Esc to jump directly to the chat input and continue typing without breaking flow.

Start New Chat

Press Ctrl + Alt + O to open a fresh chat immediately.

Delete Current Chat

Press Ctrl + Shift + Backspace to trigger the delete flow for the active chat. This opens a confirmation dialog where you can confirm deletion or cancel.

Page Info Dialog

Press Alt + / to open Page Info for active chat metadata and quick navigation context.

For Screen Reader Users

Read Last Response

Press Ctrl + Shift + Enter to hear the most recent Gemini response on demand.

New Response Announcement

PageLive announces when a new assistant response is available, so you know when new content is ready to read.

Dialog State Announcement

PageLive announces when Gemini shortcut dialogs open or close, giving event-driven status feedback during navigation.

Read Formulas and Symbols

PageLive helps screen readers read formulas and symbols in Gemini responses. Without PageLive, these items will be invisible for screen readers.

Use Cases

Jump from Page Info to Chat List

Open Page Info with Alt + /, then press Enter on the chat title control to move focus to the current chat in the side list.

Review Chat Flow with Content Map

Open ContentMap with Ctrl + M to review prompt and response summaries, then press Enter on any item to jump directly to that location in the chat history.
