PageLive

PageLive is a free Chrome extension for power users and screen reader users.

It provides keyboard shortcuts for easier navigation and actions, plus page information and event-triggered announcements to keep users informed as the page changes.

Add PageLive to Chrome (Free)

Built for Power Users

Use keyboard shortcuts to move quickly through web apps, trigger common actions, and stay in flow without constant pointer navigation.

Built for Screen Reader Users

Get useful page information on demand and receive event-triggered announcements when important updates happen.
