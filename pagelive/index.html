ChatGPT Removes Response Reading Feature

Date: 16 May 2026

ChatGPT has removed the built-in feature that allows screen readers to automatically read responses once they are fully rendered. This may have been removed unintentionally when the web application was updated.

To overcome this, you can use PageLive's feature to read the last response by pressing Ctrl + Shift + Enter.

Planned Supported Web Apps

Date: 6 May 2026

PageLive plans to support more web apps in the future, prioritizing popular web apps that still have accessibility barriers for screen reader users. To avoid users confusion if PageLive keep asking permission for each web app, we will add an extensive list of planned supported web apps in the documentation, and update the list as we make progress on development.

The development is progressing at a slow pace due to limited resources, and we appreciate users' patience and support as we continue to improve and expand PageLive's capabilities over time.
