For Screen Reader Users

Shortcut Dialog Close Announcement

When ChatGPT shortcut dialog opens, screen reader announces it. However when you close the dialog, either by pressing Escape or pressing the close button, the dialog is closed without an announcement, which can be confusing. PageLive announces when the shortcut dialog is closed, so you have clear feedback.

Fix Unreadable Keyboard Shortcuts

ChatGPT's keyboard shortcuts in the dialog can not be read by screen reader. PageLive copy append each keyboard shortcut to its corresponding label so they can be erad by screen reader. PageLive also translate some keyboard symbols to be readable, for example Enter key symbol and Escape key symbol.
