Claude New Response Reading Issue

Date: 9 March 2026

Update for screen reader users about repeated response announcements in Claude.

Who This Post Is For

This post is intended for screen reader users only, not power users.

What Changed

Before March 6, 2026, Claude did not announce newly received responses. Because of that, PageLive provided a feature to read new responses.

On March 7, I realized Claude implemented this feature. However, Claude repeatedly reads the response over and over again incrementally while the response is still being generated.

The feature to read a new response should only read finalized parts and should not repeat the same words again. When Claude keeps reading incrementally and PageLive also reads the response, it can make screen reader users confused.

Suggested Workaround

Occasionally press Esc to make the screen reader stop reading the response. Once the screen reader is no longer reading anything, press Ctrl + Shift + Enter to read the last response.
