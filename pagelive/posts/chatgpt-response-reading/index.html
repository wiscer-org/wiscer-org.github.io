ChatGPT Missing Feature to Read Received Response

Date: 16 May 2026

Who This Post Is For

This post is intended for screen reader users only, not power users.

ChatGPT Missing Feature

ChatGPT used to have a feature that allows screen reader to read responses once they are fully rendered. However the feature is no longer exist. ChatGPT only say "response complete" when the full response is received but not automatically allow screen readers to read the full response.

The Workaround

To overcome this issue you can use PageLive's feature to read the last response. This will prevent you to move focus or screen reader cursor back and forth to read and interact with the web. After you submit a prompt, just wait ChatGPT says "response complete", then press Ctrl + Shift + Enter to read the last response.
